# Cursor Integration

## Quick start

1. Copy [`drop-in/.cursorrules`](../drop-in/.cursorrules) to your project root.
2. Cursor automatically loads it for every chat in that workspace.
3. Open chat (`Cmd+L`) and ask: `Run a security audit on this codebase using the rules. Severity-tag every finding.`

## Tips

- Use `Cmd+I` (Composer) for multi-file refactors implementing fixes.
- Add the repo to "Docs" if you want Cursor to reference the full rule set, not just the compressed `.cursorrules`.
