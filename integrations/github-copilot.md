# GitHub Copilot Integration

## Quick start

1. Copy [`drop-in/copilot-instructions.md`](../drop-in/copilot-instructions.md) to `.github/copilot-instructions.md` in your repo.
2. GitHub Copilot Chat will use it for code suggestions and review.
3. In Copilot Chat: `/review` or `Audit this file for security issues.`

## Tips

- Pin frequently audited files in Copilot Chat.
- Combine with branch protection so Copilot review is mandatory before merge.
