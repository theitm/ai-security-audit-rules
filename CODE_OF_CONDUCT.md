# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be respectful. Disagree on technique, not on people.
- Defensive security only. No offensive content, no exploit walkthroughs.
- No harassment, no discrimination.
- Maintainers may remove content or contributors that violate these norms.

Report issues by opening a GitHub issue or contacting the maintainer via the GitHub profile.
